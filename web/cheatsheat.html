mijn cheatsheet

command prompt

  • dir = laat alle mappen zien
  • cd (map naam) = gaat naar de aangegeven map
  • mkdir (map naam) = maak een niewe map aan

git hub

  • git status = laat zien of er een repository is en of er aanpasing aan bestanden zijn gemaakt
  • git add (file name) = voegt de file toe aan de stageing area zo dat je commits kan maken
  • git commit = voeg een commit toe aan een file wat je naar github wil stuuren zo dat je verandringen kan zien
  • git remote add origin (repos link van github) = link de repository in github aan de map
  • git branch -M (naam) = verrandert de name van de branch
  • git push -u origin main = pushed alle commits naar github
  • git push = na git push -u origin main deze gebruiken
